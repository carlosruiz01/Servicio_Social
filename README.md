# Proyecto de Servicio Social: Juega, Aprende y Diviértete con Linux.

### Descripción del Proyecto

Proyecto de servicio social enfocado en el diseño y desarrollo de un software educativo. El objetivo principal fue crear un videojuego educativo 2D, utilizando **Godot**, para fomentar el aprendizaje lúdico de las matemáticas en estudiantes de quinto y sexto grado de primaria mediante una metodología interactiva y accesible.

- Diseño y desarrollo de un juego educativo multiplataforma (Linux y Windows), adaptable a equipos con diferentes capacidades.
- Creación de experiencias de aprendizaje interactivas que refuerzan conceptos matemáticos clave como fracciones, cálculo de precios, navegación por coordenadas y simetría.
- Desarrollo de un juego didáctico compuesto por cuatro minijuegos con una temática unificada ambientada en pizzerías.
- Diseño y animación de personajes antropomórficos originales, incluyendo zarigüeya, murciélago, cerdo, serpiente, pájaro toh y un *alux*.
